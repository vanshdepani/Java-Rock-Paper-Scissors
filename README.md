# Java-Rock-Paper-Scissors
A simple Rock Paper Scissors game in Java played via the console. The user competes against the computer, which makes random choices. This project demonstrates basic Java concepts such as conditionals, loops, arrays, user input, and random number generation.

##  Features

- The user enters one of the following choices: Rock, Paper, Scissors
- The computer randomly selects one option.
- The program compares both choices and declares.
- The user can choose to play multiple rounds.
